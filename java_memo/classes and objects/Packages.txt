Packages

패키지는 이름 충돌을 피하고 클래스에 대한 액세스를 제어하는 데 사용됩니다.
패키지는 하위 패키지와 함께 유사한 유형의 클래스로 구성된 그룹으로 정의할 수 있습니다.
Java로 패키지를 만드는 것은 매우 쉽습니다. src 디렉토리를 마우스 오른쪽 버튼으로 클릭하고 새로 만들기->패키지를 클릭하기만 하면 됩니다. 패키지 이름을 지정하고 마침을 클릭합니다.
새 패키지가 프로젝트 디렉토리에 나타납니다. 이제 해당 패키지 내에서 클래스를 이동하고 생성할 수 있습니다. Vehicle, Counter 및 Animal 클래스를 패키지 샘플로 옮겼습니다.

패키지에서 클래스를 이동/생성하면 파일 목록 상단에 다음 코드가 표시됩니다.

package samples;

클래스가 속한 패키지를 나타냅니다.
이제 메인 패키지 안에 있는 클래스를 가져와야 사용할 수 있습니다.
다음 예제에서는 샘플 패키지 의 Vehicle 클래스 를 사용하는 방법을 보여줍니다.

import samples.Vehicle;

class MyClass {
  public static void main(String[ ] args) {
    Vehicle v1 = new Vehicle();
    v1.horn();
  }
}

클래스가 패키지에 배치될 때 두 가지 주요 결과가 발생합니다. 먼저 패키지 이름이 클래스 이름의 일부가 됩니다. 둘째, 패키지 이름은 해당 클래스 파일이 있는 디렉토리 구조와 일치해야 합니다.

와일드카드를 사용하여 패키지의 모든 클래스를 가져옵니다.
예를 들어 import samples.* 는 샘플 패키지의 모든 클래스를 가져옵니다.
